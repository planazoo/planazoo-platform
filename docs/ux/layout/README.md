# UX Layout Specs (histórico)

Los archivos en `docs/ux/layout/specs/` son especificaciones históricas/técnicas de layout por pantalla.

- No son fuente normativa de reglas visuales globales.
- La fuente canónica única de UI es `docs/guias/GUIA_UI.md`.
- Si hay conflicto entre un spec de este directorio y `GUIA_UI.md`, prevalece `GUIA_UI.md`.
